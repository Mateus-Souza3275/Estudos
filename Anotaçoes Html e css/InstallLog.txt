      Html 5
📦 1. Tags de Estrutura
Usadas para estruturar a página.

Tag	Função
<html>	Define o documento HTML
<head>	Contém metadados (título, links)
<body>	Contém o conteúdo visível da página

🧱 2. Tags de Seção / Layout
Organizam o conteúdo da página.

Tag	Função

<header>	Cabeçalho da página ou seção
<nav>	        Menu de navegação
<main>	        Conteúdo principal da página
<section>	Seção de conteúdo
<article>	Conteúdo independente (como posts)
<aside>	        Conteúdo lateral (como barras laterais)
<footer>	Rodapé da página
<div>	        Container genérico para agrupar elementos

✍️ 3. Tags de Texto
Controlam títulos, parágrafos, listas, etc.

Tag	Função

<h1>–<h6>	Títulos (do maior para o menor)
<p>	Parágrafo
<br>	Quebra de linha
<hr>	Linha horizontal/divisão
<blockquote>	Citação longa

🔤 4. Tags de Formatação de Texto
Alteram o estilo do texto.

Tag	Função
<strong>	Negrito semântico (importante)
<b>	Negrito visual
<em>	Itálico semântico (ênfase)
<i>	Itálico visual
<u>	Sublinhado
<mark>	Texto marcado
<small>	Texto menor

🔗 5. Tags de Links e Mídia
Inserem links, imagens, áudio, etc.

Tag	Função
<a>	Link
<img>	Imagem
<audio>	Áudio
<video>	Vídeo
<source>	Fonte de mídia
<iframe>	Conteúdo externo incorporado

📋 6. Tags de Listas
Criam listas ordenadas e não ordenadas.

Tag	Função
<ul>	Lista não ordenada
<ol>	Lista ordenada
<li>	Item de lista
<dl>	Lista de definição
<dt>	Termo da definição
<dd>	Descrição da definição

📑 7. Tags de Formulário
Criam e organizam formulários.

Tag	Função
<form>	Define o formulário
<input>	Campo de entrada
<label>	Rótulo para campo
<textarea>	Área de texto
<button>	Botão
<select>	Menu suspenso
<option>	Opção dentro do select
<fieldset>	Agrupa elementos
<legend>	Título do grupo (fieldset)

⚙️ 8. Tags de Script e Estilo
Para incluir CSS e JavaScript.

Tag	Função
<style>	CSS embutido
<link>	Link para CSS externo
<script>	Código JavaScript
Material complementar visual studio code.

GOOGLE FONTS COM <link> (RECOMENDADO)

HTML (no <head>):

<link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">

CSS:
body {
font-family: 'Roboto', sans-serif;
}

✔️ Vantagens: Rápido, fácil, grande acervo grátis
❌ Desvantagens: Depende da internet do usuário

GOOGLE FONTS COM @import

CSS:
@import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap
');

body {
font-family: 'Roboto', sans-serif;
}

✔️ Vantagens: Simples de colar no CSS
❌ Desvantagens: Mais lento, não é recomendado

@FONT-FACE (ARQUIVOS BAIXADOS)

CSS:
@font-face {
font-family: 'MinhaFonte';
src: url('fonts/minha-fonte.woff2') format('woff2'),
url('fonts/minha-fonte.woff') format('woff');
font-weight: normal;
font-style: normal;
}

h1 {
font-family: 'MinhaFonte', sans-serif;
}

✔️ Vantagens: Independe da internet, aceita qualquer fonte
❌ Desvantagens: Precisa hospedar os arquivos, pode ser mais pesado

FONTES LOCAIS DO SISTEMA

CSS:
body {
font-family: Arial, Helvetica, sans-serif;
}

✔️ Vantagens: Muito rápido, já disponível no PC
❌ Desvantagens: Pouca variedade, pode mudar de um sistema para outro

INSTALAR FONTES NO SISTEMA (WINDOWS / MAC / LINUX)

Windows: botão direito no arquivo .ttf ou .otf → "Instalar"

MacOS: abrir no Font Book e instalar

Linux: copiar para ~/.fonts ou /usr/share/fonts

✔️ Vantagens: Disponível em qualquer aplicativo do PC (Word, Photoshop etc.)
❌ Desvantagens: Não serve para web diretamente
